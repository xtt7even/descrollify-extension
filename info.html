Information

Descrollify offers three different modes to enhance your browsing experience:

  • OFF: In this mode, the extension is inactive and does not block any short videos. This allows for an uninterrupted browsing experience where all content is accessible.
  • Limit: This mode helps you stay focused by restricting the viewing of short video content based on the number of videos you've watched. It allows you to watch some videos but blocks further viewing when you've reached your limit.
  • Focus: This mode provides the highest level of restriction by blocking all short video content and preventing accidental short video links from opening. Use this mode when you need to eliminate distractions completely and concentrate on your tasks. Optionally, you can also hide short video thumbnails.

How to Use the Modes

To switch between modes, follow these steps:

  1. Click on the extension icon in your browser's toolbar.
  2. In the dropdown menu, select the desired mode (OFF, Limit, or Focus).
  3. Adjust any additional settings to fine-tune your preferences.
  4. The extension will now operate according to the selected mode.

Options Page Manual

1. Reset Statistics

This section allows you to reset all your recorded statistics.

  • RESET STATISTICS Button: Click this button to clear all your usage statistics and start fresh.
2. Total Focus Mode
  • Hide Short Videos Thumbnails:
    • What It Does: When enabled, it hides the thumbnails of short videos to reduce distractions.
    • How to Use: Click the "Hide short videos thumbnails" button. Ensure the radio button next to it is selected.
3. Limiter Mode
  • Max Videos Allowed:
    • What It Does: Sets a limit on the number of videos you can watch before a blocker is activated.
    • How to Use:
      1. Use the number input field to set the maximum number of videos (between 1 and 30). Adjust the number by clicking the up and down arrows or typing the number directly.
  • Remove Blocker After:
    • What It Does: Sets a timer for how long the blocker will remain active before it is automatically removed.
    • How to Use:
      1. Set the desired time using the dropdown menus:
        • Select the number of hours from the "Hours" dropdown.
        • Select the number of minutes from the "Minutes" dropdown.
        • Select the number of seconds from the "Seconds" dropdown.
        • If you don't want to set any timer, use 0 for hours, minutes, and seconds.

Frequently Asked Questions

Is the extension compatible with all browsers?
Currently, the extension is compatible with popular browsers like Google Chrome and Microsoft Edge. It's planned to expand compatibility to other browsers.
Does it block short videos from all social media?
Currently, this extension is only compatible with YouTube shorts. I'm working on implementing wider compatibility with other social media platforms.
Does Descrollify access my watch or browser history or save sensitive information?
No, Descrollify does not collect any data from your watch or browser history. It generates its own data as you scroll through different short videos to create statistical information. Moreover, all the data is stored locally on your computer. The extension does not save any video links or other sensitive data.
